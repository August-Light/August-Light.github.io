# 置顶：RLC 电路与力学的类比

|  | 弹簧 | 机械转动 | RLC 电路 |
| - | - | - | - |
| 质量 | $m$ | $I$ | $L$ |
| 位移 | $x$ | $\theta$ | $Q$ |
| 速度 | $v = \dot x$ | $\omega = \dot \theta$ | $I = \dot Q$ |
| 动能 | $\mathrm{KE} = \frac 1 2 m v^2$ | $\mathrm{KE} = \frac 1 2 I \omega^2$ | $U_{\text{ind}} = \frac 1 2 L I^2$ |
| 动量 | $p = mv$ | $L = I \omega$ | $\Phi_B = LI$ |
| 力 | $F$ | $\tau$ | $\Delta V$ |
| 弹性系数 | $k$ | $\kappa$ | $\frac 1 C$ |
| Hooke 定律 | $F = kx$ | $\tau = \kappa \theta$ | $\Delta V = \frac 1 C Q$ |
| 弹性势能 | $U = \frac 1 2 k x^2$ | $U = \frac 1 2 \kappa \theta^2$ | $U_{\text{cap}} = \frac 1 2 \frac 1 C Q^2$
| 阻尼系数 | $b$ | $b$ | $R$ |

# 静电场

$$\boxed{
    F = \frac 1 {4 \pi \varepsilon_0} \frac {\lvert Q_1 \rvert \lvert Q_2 \rvert} {r^2}
}$$

$$\boxed{
    \bm E = \frac {\bm F} q \iff V = \frac U q
}$$

$$\bm E = - \nabla V \iff \Delta V = - \int \bm E \cdot d \bm l$$

$$\boxed{
    \oiint \bm E \cdot d \bm A = \frac Q {\varepsilon_0}
}$$

$$\textcolor{lightgreen}{
    dW = (\Delta V) dQ
} \iff P = (\Delta V) I$$

## 点电荷

电势：

$$\textcolor{lightgreen}{
    V(r) = \frac Q {4 \pi \varepsilon_0 r}
}$$

## 无穷大带电平板

电场：

$$\textcolor{lightgreen}{
    E = \frac \sigma {2 \varepsilon_0}
}$$

**以板上为电势零点**，电势：

$$\textcolor{lightgreen}{
    V(d) = - E d =  - \frac \sigma {2 \varepsilon_0} d
}$$

# 电容器

$$\boxed{
    C = \frac Q {\Delta V}
} \textcolor{gray}{\implies \frac 1 C = \frac {\Delta V} Q}$$

$$U \textcolor{gray}{= \frac 1 2 \frac 1 C Q^2} = \frac 1 2 Q \Delta V = \frac 1 2 C (\Delta V)^2$$

## 串并联

串联：

$$\frac 1 C = \sum \frac 1 {C_i}$$

并联：

$$\textcolor{gray}{\frac 1 {1 / C} = \sum \frac 1 {1 / C_i} \implies} C = \sum C_i$$

## 平行板电容器

$$V = E d$$

$$\textcolor{lightgreen}{
    E = \frac \sigma {\varepsilon}
}$$

$$\textcolor{gray}{A = \frac Q \sigma \implies} C = \frac {A \varepsilon} d$$

$\varepsilon$ 在真空中等于 $\varepsilon_0$，在其它介质中

$$\textcolor{gray}{\varepsilon = \kappa \varepsilon_0}, \quad \textcolor{lightgreen}{\kappa > 1}$$

因此

$$\boxed{\textcolor{lightgreen}{
    C = \kappa C_0
}}$$

$$C = \frac {\kappa A \varepsilon_0} d$$

# 电路

$$\boxed{
    I = \frac {dQ} {dt}
}$$

$$\boxed{
    I = \int \bm J \cdot d\bm A
}$$

## 电阻与 Ohm 定律

$$\boxed{
    \Delta V = I R \iff \bm E = \rho \bm J
}$$

柱状导体的电阻：

$$R = \frac {\rho L} A$$

电导率的定义：

$$\boxed{\textcolor{lightgreen}{
    \sigma = \frac 1 \rho
}}$$

### 串并联

串联：

$$R = \sum R_i$$

并联：

$$\frac 1 R = \sum \frac 1 {R_i}$$

## 电流

载流子与漂移速度：

$$\boxed{
    \bm J = n e \bm v_d
}$$

# 静磁场

|  | Produce $\bm B$ | Respond to $\bm B$ |
| :-: | :-: | :-: |
| Moving Charge | $\bm B = \frac {\mu_0} {4\pi} \frac {\textcolor{goldenrod}{q \bm v} \times \hat{r}} {r^2}$ | $\bm F = \textcolor{goldenrod}{q \bm v}  \times \bm B$ |
| Differential Length of Wire | $d\bm B = \frac {\mu_0} {4\pi} \frac {\textcolor{limegreen}{I d\bm l} \times \hat{r}} {r^2}$ | $d\bm F = \textcolor{limegreen}{I d\bm l} \times \bm B$ |

$$\boxed{
    \bm F = q (\bm E + \bm v \times \bm B)
}$$

$$\boxed{
    \oiint \bm B \cdot d \bm A = 0
}$$

$$\boxed{
    \oint \bm B \cdot d \bm l = \mu_0 I
}$$

## 无限长直导线

$$\textcolor{lightgreen}{
    B = \frac {\mu_0 I} {2\pi r}
}$$

无限长直导线间作用力：

$$\textcolor{lightgreen}{
    \frac F l = \frac {\mu_0 I_1 I_2} {2\pi r}
}$$

## 通电螺线管

$$\textcolor{lightgreen}{B = \begin{cases}
    \mu_0 n I, & \quad r < R \\
    0, & \quad r > R
\end{cases}}$$

# 磁生电

$$\boxed{
    \Phi_B = \iint \bm B \cdot d \bm A
}$$

$$\boxed{
    \mathcal E = - \frac {d \Phi_B} {dt}
}$$

## 电感器

$$\boxed{\textcolor{lightgreen}{
    \Phi_B = LI
}} \iff \mathcal E = - L \frac {dI} {dt}$$

$$U = \frac 1 2 L I^2$$

### 串并联

串联：

$$\textcolor{lightgreen}{
    L = \sum L_i
}$$

并联：

$$\textcolor{lightgreen}{
    \frac 1 L = \sum \frac 1 {L_i}
}$$

# Ampere-Maxwell 定律

$$\boxed{
    \oint \bm B \cdot d \bm l = \mu_0 I + \mu_0 \varepsilon_0 \frac {d \Phi_E} {dt}
}$$

# 特殊电路

## RC & RL 电路

$$\begin{aligned}
    x(t) &= x_\infty + (x_0 - x_\infty) e^{- \frac t \tau} \\
    &= x_0 e^{- \frac t \tau} + x_\infty \left( 1 - e^{- \frac t \tau} \right)
\end{aligned}$$

### RC 电路

$$R \frac {dQ} {dt} + \frac Q C = 0$$

$$\boxed{\textcolor{lightgreen}{
    \tau = RC
}}$$

| 物理量 | 充电 | 放电 |
| - | - | - |
| $Q$ | $Q = C \mathcal E \left( 1 - e^{- \frac t \tau} \right)$ | $Q = Q_0 e^{- \frac t \tau}$ |
| $I = \frac {dQ} {dt}$ | $I = \frac {\mathcal E} R e^{- \frac t \tau}$ | $I = I_0 e^{- \frac t \tau}$ |

### RL 电路

$$- L \frac {dI} {dt} - RI = 0$$

$$\boxed{\textcolor{lightgreen}{
    \tau = \frac L R
}}$$

| 物理量 | 充电 | 放电 |
| - | - | - |
| $I$ | $I = \frac {\mathcal E} R \left( 1 - e^{- \frac t \tau} \right)$ | $I = I_0 e^{- \frac t \tau}$ |
| $V = L \frac {dI} {dt}$ | $V = \mathcal E e^{- \frac t \tau}$ | $V = V_0 e^{- \frac t \tau}$ |

## LC 电路

$$\frac {d^2Q} {dt^2} + \frac 1 {LC} Q = 0$$

$$\textcolor{lightgreen}{
    \omega = \frac 1 {\sqrt{LC}}
}$$

$$Q = K \sin(\omega t + \varphi) \implies I = - \frac {dQ} {dt} = - K \omega \cos(\omega t + \varphi)$$
