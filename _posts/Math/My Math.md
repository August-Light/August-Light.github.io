# Trigonometric Function

- $\tan a + \tan b$ 和 $\cot a + \cot b$
  - $\tan a + \tan b = \frac {\sin(a + b)} {\cos a \cos b}$
  - $\cot a + \cot b = \frac {\sin(a + b)} {\sin a \sin b}$
