# [2013 AIME II #15](https://artofproblemsolving.com/wiki/index.php?title=2013_AIME_II_Problems/Problem_15)

> 已知 $A,B,C$ 是三角形的三个角。
>
> $$\begin{cases}
>     \cos^2 A + \cos^2 B + 2 \sin A \sin B \cos C = \frac {15} 8 \\
>     \cos^2 B + \cos^2 C + 2 \sin B \sin C \cos A = \frac {14} 9 \\
>     \cos^2 C + \cos^2 A + 2 \sin C \sin A \cos B = ans \\
> \end{cases}$$
>
> 求 $ans$。

我们尝试化简：

$$\begin{aligned}
    & \cos^2 A + \cos^2 B + 2 \sin A \sin B \cos C \\
    =& 2 - (\sin^2 A + \sin^2 B - 2 \sin A \sin B \cos C)
\end{aligned}$$

显然是余弦定理的表达式（两边为 $\sin A, \sin B$，夹角为 $C$）。肯定要数形结合了。

考虑构造一个三边分别为 $\sin A, \sin B, \sin C$ 的三角形。根据正弦定理 $a:b:c = \sin A : \sin B : \sin C$，这个三角形和原三角形恰好是相似的，即三个角依然是 $A,B,C$。

根据刚才的余弦定理计算出：

$$\sin C = \sqrt {\frac 1 8}, \sin A = \sqrt{\frac 4 9}$$

此时再回到最后一个式子

$$\sin^2 C + \sin^2 A + 2 \sin C \sin A \textcolor{red}{\cos B} = \textcolor{red}{\sin^2 B} = 2 - ans$$

这是一个关于 $\cos B$ 的二次方程，最终解出

$$ans = \boxed{\frac {111 - 4 \sqrt {35}} {72}}$$
