# 简单代数

## $\sqrt{a \pm \sqrt b}$ 化简

$$\Delta = a^2 - b$$

$$\sqrt{a \pm \sqrt b} = \sqrt{\frac {a + \sqrt \Delta} 2} \pm \sqrt{\frac {a - \sqrt \Delta} 2}$$

核心思想：设 $\sqrt{a \pm \sqrt b} = \sqrt x \pm \sqrt y$，两边平方后得到 $x + y$ 和 $xy$ 的值，解出 $x,y$ 即可。

# 三角函数

$$\begin{cases}
    1 + \sin x = 2 \sin^2 \left( \frac x 2 + \frac \pi 4 \right) \\
    1 - \sin x = 2 \cos^2 \left( \frac x 2 + \frac \pi 4 \right) \\
    1 + \cos x = 2 \cos^2 \frac x 2 \\
    1 - \cos x = 2 \sin^2 \frac x 2 \\
\end{cases}$$

这组公式似乎是可以推广的，$A \pm B \sin x$ 和 $A \pm B \cos x$ 似乎都可以表示成平方的形式。

# 双曲函数

## 导数表

$$\begin{cases}
    \sinh' x = \cosh x \\
    \cosh' x = \sinh x \\
    \tanh' x = \operatorname{sech}^2 x \\
\end{cases}$$

$$\begin{cases}
    \operatorname{arsinh}' x = \frac 1 {\sqrt{x^2 + 1}} \\
    \operatorname{arcosh}' x = \frac 1 {\sqrt{x^2 - 1}} \\
    \operatorname{artanh}' x = \frac 1 {1 - x^2} \\
\end{cases}$$

## 与三角函数转换

$$\begin{cases}
    \sin(ix) = -i \sinh x \\
    \cos(ix) = \cosh x \\
\end{cases}$$

例：

$$\begin{aligned}
    \cos(ix + iy) &= \cos(ix) \cos(iy) - \sin(ix) \sin(iy) \\
    \cosh(x + y) &= \cosh x \cosh y - (-i \sinh x) (-i \sinh y) \\
    &= \cosh x \cosh y + \sinh x \sinh y \\
\end{aligned}$$

一个用于检查的小技巧：原三角恒等式中含有 $2$ 个 $\sin$（或者 $\tan$）的项要变号。

# 积分

## 换元法

第一类换元法：

$$u = u(x)$$

$$\int_a^b f(u(x)) \frac {du} {dx} dx = \int_{u(a)}^{u(b)} f(u) du$$

第二类换元法：

$$x = x(t)$$

$$\int_a^b f(x) dx = \int_{x^{-1}(a)}^{x^{-1}(b)} f(x(t)) \frac {dx} {dt} dt$$

---

$$\int f(ax) dx = \frac 1 a F(ax) + C$$

$$\int \frac 1 {a + b x^2} dx = \frac 1 {\sqrt{ab}} \arctan\left( \sqrt{\frac b a} x \right) + C, \quad a,b > 0$$

$$\int \frac 1 {a - b x^2} dx = \frac 1 {\sqrt{ab}} \operatorname{artanh}\left( \sqrt{\frac b a} x \right) + C, \quad a,b > 0, \lvert x \rvert < \sqrt{\frac a b}$$

## 区间再现

- 反转区间：

$$\int_a^b f(x) dx = \int_a^b f(a + b - x) dx$$

- 首尾相加：

$$\int_a^b f(x) dx = \frac 1 2 \int_a^b (f(x) + f(a + b - x)) dx$$

### 有关三角函数的区间再现

- $[0, \pi]$ 上关于 $\sin x$ 的积分，消除 $x$：

$$\int_0^\pi x f(\sin x) dx = \frac \pi 2 \int_0^\pi f(\sin x) dx$$

- $[0, \frac \pi 2]$ 上 $\sin$ 与 $\cos$ 可以互换：

$$\int_0^{\frac \pi 2} f(\sin x, \cos x) dx = \int_0^{\frac \pi 2} f(\cos x, \sin x) dx$$

### 倒代换区间再现

- 反转区间：

$$\int_{\frac 1 a}^a f(x) dx = \int_{\frac 1 a}^a \frac 1 {x^2} f \left( \frac 1 x \right) dx$$

- 首尾相加：

$$\int_{\frac 1 a}^a f(x) dx = \frac 1 2 \int_{\frac 1 a}^a \left( f(x) + \frac 1 {x^2} f \left( \frac 1 x \right) \right) dx$$

## 套路积分

$P,Q$ 是任意多项式。

### 有理函数：$\int \frac {P(x)} {Q(x)} dx$

$$\frac {P(x)} {Q(x)} = \sum \frac {P_i(x)} {(x - a_i)^{n_i}} + \sum \frac {P_j(x)} {(x^2 + b_j x + c_j)^{n_j}}$$

$$\int \frac {P(x)} {(x - a)^n} dx = \int \sum_{k=1}^n \frac {A_k} {(x - a)^k} dx$$

$$\int \frac {P(x)} {(x^2 + bx + c)^n} dx = \int \sum_{k=1}^n \frac {A_k x + B_k} {(x^2 + bx + c)^k} dx$$

$$\int \frac {Ax + B} {(x^2 + bx + c)^n} dx = \int \frac {\frac A 2 (2x + b) + (B - \frac A 2 b)} {(x^2 + bx + c)^n} dx$$

### 多项式乘 $e^x$：$\int P(x) e^x dx$

$$\int P(x) e^x dx = P(x) e^x - \int P'(x) e^x dx$$

$$\int x^n e^x dx = x^n e^x - n \int x^{n-1} e^x dx$$

### $e^{x^2}$ 相关：$\int x^n e^{x^2} dx$

$$\int x^{2n+1} e^{x^2} dx \xlongequal{u=x^2} \frac 1 2 \int u^n e^u du$$

$$\int x^{2n} e^{x^2} dx \ \text{不可做}$$

但注意偶次项的线性组合不一定不可做！例如 $\int (2x^4 + 3x^2) e^{x^2} dx = x^3 e^{x^2} + C$。

### $\sin$ 与 $\cos$ 的幂：$\int \sin^n x \cos^m x dx$

$$\int \sin^{2n+1}(x) \cos^m(x) dx \xlongequal{u = \cos x} - \int (1 - u^2)^n u^m du$$

$$\int \sin^n(x) \cos^{2m+1}(x) dx \xlongequal{u = \sin x} \int u^n (1 - u^2)^m du$$

$$\int \sin^{2n}(x) \cos^{2m}(x) dx = \int \left( \frac {1 - \cos 2x} 2 \right)^n \left( \frac {1 + \cos 2x} 2 \right)^m dx$$

Alternative method（STEP III 2025 #1）：Let $z = e^{i \theta}$,

$$\int \sin^n(x) \cos^m(x) dx = \int \left( \frac {z - z^{-1}} {2i} \right)^n \left( \frac {z + z^{-1}} 2 \right)^m dx$$

### $\tan$ 的幂：$\int \tan^n x dx$

$$\int \tan x dx = - \ln \lvert \cos x \rvert + C$$

$$\begin{aligned}
    & \int \tan^n x dx \\
    =& \int \tan^{n-2} x (\sec^2 x - 1) dx \\
    =& \frac {\tan^{n-1} x} {n-1} - \int \tan^{n-2} x dx \\
\end{aligned}$$

$\int \cot^n x dx$ 同理。

### $\sec$ 的幂：$\int \sec^n x dx$

$$\int \sec x dx = \ln \lvert \sec x + \tan x \rvert + C$$

$$\int \sec^2 x dx = \tan x + C$$

$$\begin{aligned}
    & \int \sec^n x dx \\
    =& \int \sec^{n-2} x \times \sec^2 x dx \\
    =& \sec^{n-2} x \tan x - (n-2) \int \sec^{n-2} x \tan^2 x dx \\
    =& \sec^{n-2} x \tan x - (n-2) \left( \int \sec^n x dx - \int \sec^{n-2} x dx \right) \\
\end{aligned}$$

$$\int \sec^n x dx = \frac{\sec^{n-2}x \tan x}{n-1} + \frac{n-2}{n-1} \int \sec^{n-2} x dx$$

$\int \csc^n x dx$ 同理。

$$\det \begin{bmatrix} a & b \\ c & d \end{bmatrix} = ad - bc$$
