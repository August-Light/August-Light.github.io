# Supervised / Unsupervised / Reinforcement learning

- Supervised learning
  - 使用带有标签的数据进行训练，即每条数据都有对应的正确答案
  - 学习从输入到输出的映射关系，以便对新数据进行预测。
- Unsupervised learning
  - 使用无标签的数据进行训练，算法自行发现数据中的隐藏结构或模式。
  - 探索数据内在的分布和 grouping。
- Reinforcement learning
  - 通过智能体与环境的交互进行学习。智能体执行动作，环境返回奖励或惩罚。
  - 学习一套策略，使得累积奖励最大化。

# Regression, Classification, Clustering

- Regression：预测一个连续值的输出。
- Classification：预测一个离散值的输出。
- Clustering：一种无监督学习任务，将物理或抽象对象的集合分组为由类似的对象组成的多个类。

# Linear regression, Logistic regression

- Linear regression：试图找到一条直线（或超平面）来最佳地拟合数据点，使得预测值与真实值之间的均方误差最小。
- Logistic regression：虽然名字里有“回归”，但它实际上是用于二分类的算法。它将线性回归的输出通过一个 Sigmoid 函数，映射到 0 到 1 之间，代表样本属于某一类别的概率。

# Module evaluation

Confusion matrix：

|  | 预测为正类 P | 预测为负类 N |
| - | - | - |
| 真实为正类 | $\textcolor{#03C03C}{T}P$ | $\textcolor{#E32636}{F}N$ |
| 真实为负类 | $\textcolor{#E32636}{F}P$ | $\textcolor{#03C03C}{T}N$ |

整个表格的正确率（准确率）：

$$\text{Accuracy} = \frac {\textcolor{#03C03C}{T}P + \textcolor{#03C03C}{T}N} {TP + TN + FP + FN}$$

第一列的正确率（精确率）：

$$\text{Precision} = \frac {T\textcolor{#1E90FF}{P}} {T\textcolor{#1E90FF}{P} + F\textcolor{#1E90FF}{P}}$$

第一行的正确率（召回率）：

$$\text{Recall} = \frac {TP} {TP + FN}$$

两者的调和平均数：

$$\text{F1-score} = \frac 2 {\frac 1 {\text{Precision}} + \frac 1 {\text{Recall}}}$$

Cross validation：将数据集分成 K 份，轮流将其中 K-1 份做训练，1 份做验证，最终取 K 次结果的均值作为评估结果。有效避免过拟合，提高模型泛化能力。

# Underfitting, Overfitting, Regularization

- Underfitting：模型过于简单，无法捕捉数据中的潜在规律，在训练集上表现就很差。
- Overfitting：模型过于复杂，不仅学习了数据中的规律，还“死记硬背”了训练数据中的噪声和细节，导致在训练集上表现很好，但在新数据上表现很差。
- Regularization：用于防止过拟合的技术。通过在损失函数中加入一个惩罚项，限制模型的复杂度。

# Ensemble learning

- Ensemble learning：通过结合多个弱学习器（基础模型）来构建一个更强的学习器
  - Bagging
    - 对数据进行有放回的多次抽样，训练多个独立的模型，最后通过投票或平均进行预测。
    - Random forest
  - Boosting
    - 按顺序训练模型，后一个模型会重点关注前一个模型预测错误的样本，不断“纠错”。
    - Gradient boosting

# Training set, Testing set, Validation set

- Training set：用于训练模型，让模型学习参数。
- Validation set：用于模型选择和调参，评估不同模型或超参数的表现，帮助防止过拟合。
- Testing set：在模型最终确定后，用于最终评估模型泛化能力的“最终考试”，模型在整个训练过程中绝对不能接触到测试集。

# Perceptron, NN (Neural Network)

- Perceptron：最简单的神经网络形式，模拟单个神经元。接收多个输入，加权求和后通过一个阶跃函数输出。
- NN：由大量神经元（感知机）相互连接而成的网络。通常包含输入层、若干隐藏层和输出层。

# Backpropagation

- 正向传播：输入数据从输入层经过各隐藏层计算，最终到达输出层，得到预测结果。
- 计算损失：计算预测结果与真实标签之间的误差。
- 反向传播：利用链式法则，将误差从输出层反向传播回输入层，计算损失函数关于每个权重的梯度。
- 更新权重：根据计算出的梯度，使用梯度下降法更新网络中的所有权重。

# Gradient Descent

一种优化算法，通过沿着目标函数（损失函数）梯度的反方向更新模型参数，来最小化损失函数。

# Local optima, Global optima

- Global optima：在整个参数空间中，损失函数达到全局最小值点。
- Local optima：在参数空间的某个局部区域，损失函数达到最小值，但不是全局最低的。梯度下降容易陷入局部最优。

# CNN (Convolutional Neural Network)

一种专门处理具有类似网格结构数据的神经网络，最常用于处理图像。其核心思想是利用卷积核自动提取图像的空间特征，同时通过参数共享和池化大大减少参数量。

# Convolutional layer, Pooling layer, Fully connected layer

- Convolutional layer
  - 作用：提取特征。
  - 计算过程：一个可学习的卷积核（滤波器）在输入图像上滑动，与覆盖的区域进行点积运算，生成一张特征图（feature map）。
- Pooling layer
  - 作用：降低特征图的维度（下采样），减少计算量，防止过拟合，并增强特征的平移不变性。
  - 计算过程：通常取一个窗口内的最大值（Max pooling）或平均值（Average pooling）。
- Fully connected layer
  - 作用：将经过卷积和池化提取到的高维特征进行整合，映射到最终的输出（如类别概率）。
  - 计算过程：传统神经网络的矩阵乘法，将输入展平后与权重相乘并加偏置。

# Decision function, Activation function, Loss function

- Decision function
  - 作用：在分类问题中，根据模型输出的分数（如概率）来决定最终的类别归属。例如，逻辑回归中通常设定阈值为 0.5。
- Activation function
  - 作用：引入非线性。如果没有激活函数，无论神经网络有多少层，都只是线性变换的叠加，无法解决复杂问题。常见的如 ReLU、Sigmoid、Tanh。
- Loss function
  - 作用：量化模型预测值与真实值之间的不一致程度。损失函数值越小，模型性能越好。它是模型优化的目标。例如回归问题的均方误差，分类问题的交叉熵损失。

# Algorithm bias

- 含义：算法产生系统性的、不公平的歧视或偏见，对某些特定群体造成不公。
- 社会影响及案例：可能导致招聘歧视（如偏向男性）、司法量刑偏差（如对少数族裔判罚更重）、金融信贷歧视等，加剧社会不平等。
- 减少偏见的方法：
  - 技术手段：使用多样化、有代表性的数据集；开发和应用公平性指标（如均等化几率）；研究可解释性 AI，理解算法决策原因；在模型训练中引入公平性约束。
  - 政策手段：建立算法伦理审查委员会；制定相关法律法规，明确算法问责制；推动行业标准制定；提高算法透明度，接受公众监督。
