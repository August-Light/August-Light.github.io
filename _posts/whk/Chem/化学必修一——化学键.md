- **化学键**：物质中近邻的原子或离子（**不包括分子**）之间的强烈的相互作用。
  - 关于分子：$\ce{He, Ne, Ar, Kr}$ 算单原子**分子**；金属单质**不是分子**。
- 化学反应的实质是：反应物中旧化学键的断裂和生成物中新化学键的形成。
- 破坏化学键吸收能量，形成化学键放出能量。
- 成键动力：使外层电子达到稳定结构。
- 化学键的分类：**离子键**、**共价键**、金属键。接下来聊聊前两种。

# 离子键

- 离子键：正负离子由于**静电作用**形成的化学键。
- **离子化合物**：以离子键结合的化合物
  - 不一定只有离子键，也可以有共价键！
  - 通常由活泼金属和活泼非金属组成。
- 常见离子化合物
  - **强碱**
  - 绝大多数**盐**
    - 常见例外：$\ce{AlCl3}$，是共价化合物。
  - **金属（过）氧化物**
- 常见混淆：**只含有非金属元素**的化合物可以是离子化合物。
  - 铵来代替金属其实就行了，比如 $\ce{(NH4)2SO4}$。

## 电子式（离子键）

> 由于无法打出周围围一圈电子的效果，我用 $(n)$ 表示周围围着 $n$ 个电子。

离子化合物的形成过程可以用电子式表示。以 $\ce{NaCl}$ 为例：

$$\ce{Na(1) + Cl(7) -> Na+ [Cl(8)]-}$$

把离子直接并列写在一起即可。

注意这个式子写**箭头**，不写等号。

# 共价键

- 共价键：**非金属元素**的**原子**通过**共用电子对**形成的化学键。
- 具有**方向性**、**饱和性**。
  - 方向性就是成键出来的东西角度是确定的。
  - 饱和性就是一个原子能成的共价键数量有限。
- 常见物质
  - 非金属单质 e.g. $\ce{O2, N2, Cl2}$
  - **共价化合物**
    - **非金属氧化物** e.g. $\ce{CO2}$
    - **气态氢化物** e.g. $\ce{NH3}$
    - **酸** e.g. $\ce{HCl, H2S}$
  - 复杂离子
    - $\ce{CO3^2-, NO3-, NH4+, SO4^2-, OH-}$
  - 大多数有机化合物
- 有球棍模型和空间填充模型两种常见的样子。

## 电子式（共价键）

> 由于无法打出周围围一圈电子的效果，我用 $(n)$ 表示周围围着 $n$ 个电子，用 $\textcolor{red}{[n]}$ 代表共用 $n$ **对**电子。

$$\ce{H(1) + Cl(7) -> H\textcolor{red}{[1]}Cl(6)}$$

注意这个式子写**箭头**，不写等号。

常见例子：

| 物质 | 电子式 |
| - | - |
| $\ce{NaOH}$ | $\ce{Na+ [O(6)\textcolor{red}{[1]} H]-}$ |
| $\ce{O_2^2-}$ | $\ce{[O(6) \textcolor{red}{[1]} O(6)]^2-}$
| $\ce{N2}$ | $\ce{N(2) \textcolor{red}{[3]} N(2)}$ |
