# 金属

## 金属的物理性质

- 通性 -> 由自由电子决定
  - 金属光泽（饰品）
  - 延展性（金属片、金属丝）
  - 导电性（电线）
  - 导热性（炊具）
- 特性 -> 由金属正离子决定
  - 熔沸点（最高 $\ce{W}$，最低 $\ce{Hg}$）
  - 硬度（最大 $\ce{Cr}$）
  - 密度（最小 $\ce{Li}$）

## 合金

- 合金：一种金属单质 + 其它单质 组成的混合物（不是化合物），具有金属特性。
- 更优良的性能
  - 硬度更大（钢铁）
  - 熔点更低（保险丝）
  - 密度更小（航天器）
  - 更耐腐蚀（不锈钢）

## 结构

- 原子结构
  - 原子半径较大
  - 最外层电子数少
- 元素周期表位置：主族、副族+VIII 族
- 化学性质
  - 相似性：易失电子（还原性）

# $\ce{Na}$

- $\ce{Na}$ 银白色
- $\ce{Na2O}$ 白色
- $\ce{Na2O2}$ 淡黄色

## $\ce{Na + O2}$

常温：

$$\ce{4 Na + O2 -> 2 Na2O}$$

空气中钠表面的银白色金属光泽很快变暗。但是这个和 $\ce{Fe, Al}$ 等元素的钝化反应不一样，ChatGPT 说它挡不住进一步反应。

加热：

$$\ce{2 Na + O2 ->[\triangle] \overset{+1}{Na}_2\overset{\textcolor{red}{-1}}{O}_2}$$

与此同时，$\ce{Na2O}$ 也可以和 $\ce{O2}$ 反应，同样生成 $\ce{Na2O2}$：

$$\ce{2 Na2O + O2 ->[\triangle] 2 Na2O2}$$

## $\ce{Na}$ 与其它非金属

剧烈燃烧，放热，黄色火焰，白烟：

$$\ce{2Na + Cl2 ->[点燃] 2NaCl}$$

研磨时发生爆炸

$$\ce{2Na + S ->[\triangle 或研磨] Na2S}$$

$$\ce{2Na + H2 ->[高温] 2NaH}$$

## $\ce{Na + H2O}$

$$\boxed{
    \ce{2Na + 2H2O -> 2NaOH + H2 ^}
}$$

离子方程式：

$$\ce{2Na + 2H2O -> 2Na+ + 2OH- + H2 ^}$$

**现象：浮熔游响红**：
- **浮**——钠浮在水面：钠的密度比水小
- **熔**——钠熔化成闪亮的小球：钠的熔点较低
- **游**——钠小球在水面四处游动：产生的气体推动小球做无规则运动
- **响**——并发出“嘶嘶”声：反应剧烈
- **红**——滴加酚酞试液后溶液变红：钠与水反应有碱性物质生成

## $\ce{Na}$ 与盐溶液反应

|  | $\ce{Na}$ |
| - | - |
| 酸 | $\begin{cases} \text{first} & \ce{2 Na + 2 HCl -> 2 NaCl + H2 ^} \\ \text{second} & \ce{2 Na + 2 H2O -> 2 NaOH + H2 ^} \\ \end{cases}$ |
| 盐溶液 | $\begin{cases} \text{first} & \ce{2 Na + 2 H2O -> 2 NaOH + H2 ^} \\ \text{second} & \ce{2 NaOH + CuCl2 -> Cu(OH2) v + 2 NaCl} \\ \end{cases}$ |

例子：$\ce{Na}$ 扔到 $\ce{FeSO4}$ 溶液，产生 $\ce{Fe(OH)2}$。$\ce{Fe(OH)2}$ 为白色沉淀；与氧气反应变为灰绿色最后变成红褐色。

# 金属的冶炼

$$\ce{M^{n+} +} n \ce{ e- -> M}$$

金属越活泼 -> $\ce{M^{n+}}$ 的氧化性（得电子能力）越弱 -> 越难被冶炼

| 方法 | 金属 |
| - | - |
| 热分解法 | $\ce{Ag}$ <br> $\ce{Hg}$ |
| 热还原法 | $\ce{Cu}$ <br> $\ce{Pb}$ <br> $\ce{Sn}$ <br> $\ce{Fe}$ <br> $\ce{Zn}$ |
| 电解法 | $\ce{Al}$ <br> $\ce{Mg}$ <br> $\ce{Na}$ <br> $\ce{Ca}$ <br> $\ce{K}$ |

## 热分解法（$\ce{Ag, Hg}$）

$$\ce{AgS + O2 ->[焙烧] Ag + SO2}$$

$$\ce{HgS + O2 ->[焙烧] Hg + SO2}$$

$$\ce{2 HgO ->[\triangle] 2 Hg + O2 ^}$$

$\ce{AgS, HgO}$ 都是既作氧化剂又作还原剂。

## 热还原法（$\ce{Cu, Fe}$）

需要额外加入还原剂。

$$\ce{2 CuO + C ->[高温] 2 Cu + CO2 ^}$$

$$\ce{Fe2O3 + 3CO ->[高温] 2Fe + 3CO2}$$

较活泼金属可以在加热过程中利用还原剂换元其化合物制得。

常见还原剂：$\ce{H2}$、焦炭、$\ce{CO}$、**活泼金属**。

### 湿法炼铜

|  | 湿法炼铜 | 火法炼铜 |
| - | - | - |
| 方程式 | $\ce{Fe + CuSO4 -> Cu + FeSO4}$ | $\begin{cases} \ce{Cu2(OH)2CO3 ->[\triangle] 2CuO + H2O + CO2 ^} \\ \ce{2 CuO + C ->[高温] 2 Cu + CO2 ^} \\ \end{cases}$ |
| 缺点 | 需要大量铁 | 工艺复杂、能耗大、纯度不高 |

## 电解法（$\ce{Na, Al}$）

$$\ce{2 NaCl (熔融) ->[通直流电] 2 Na + Cl2}$$

$$\ce{2 Al2O3 ->[通直流电][助熔剂] 4Al + 3O2 ^}$$
