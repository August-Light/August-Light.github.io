# 氮气的基本性质

- 无色无味气体
- 密度略小于空气
- 氮氮三键非常稳定，因此常温常压下 $\ce{N2}$ 化学性质非常稳定。
  - 用作保护气。

# 固氮

将空气中的游离态氮转化为化合态氮的过程称为固氮。

$$\ce{N2 + O2 ->[放电] 2NO}$$

氮气与氧气生成 $\ce{NO}$：
- 无色
- 有毒
- 难溶于水
- 收集只能用排水法

$$\ce{2NO + O2 -> 2 NO2}$$

一氧化氮与氧气生成 $\ce{NO2}$：
- 红棕色
- 有刺激性气味
- 有毒
- 易溶于水
- 收集只能用向上排空气法

$$\ce{3 NO2 + H2O -> 2 HNO3 + NO}$$

$\ce{NO2}$ 遇到水生成硝酸。这可以造成**硝酸型酸雨**。

$\ce{NO, NO2}$ 会造成**光化学烟雾**；$\ce{NO2}$ 可以把 $\ce{I-}$ 氧化成 $\ce{I2}$，因此可以使**湿润的 $\ce{KI}$ 试纸变蓝**。

这两个反应可以合起来写成：

$$\begin{cases}
    \ce{4 NO2 + 2 H2O + O2 -> 4 HNO3} \\
    \ce{4 NO + 3 O2 + 2 H2O -> 4 HNO3} \\
\end{cases}$$

人工固氮，通过工业合成氨实现。

$$\ce{N2 + 3 H2 <=>[高温、高压][催化剂] 2 NH3}$$

# $\ce{NH3}$

## 物理性质

- 无色气体
- 有刺激性气味
- 冷却或加压易液化
- 汽化时要吸收大量的热，使周围温度急剧下降
  - 作制冷剂
- 极易溶于水，$1:700$
- $\ce{NH3}$ 的**检验**：唯一能使红色石蕊试液变蓝的气体。

## $\ce{NH3 * H2O}$

$$\ce{NH3 + H2O <=> NH3 * H2O}$$

$$\ce{NH3 * H2O ->[\triangle] NH3 ^ + H2O}$$

是弱碱，是弱电解质：

$$\ce{NH3 * H2O <=> NH4+ + OH-}$$

### 和 $\ce{SO2}$ 反应

氨过量：

$$\ce{2 NH3 * H2O + SO2 -> (NH4)2SO4 + H2O}$$

$\ce{SO2}$ 过量：

$$\ce{NH3 * H2O + SO2 -> NH4HSO3}$$

### 和酸反应

$$\ce{NH3 + H+ -> NH4+}$$

$$\ce{NH3 * H2O + H+ -> NH4+ + H2O}$$

常见反应：

$$\ce{NH3 (g) + HCl (g) -> NH4Cl (s)}$$

## 燃烧

$$\ce{4NH3 + 3O2 ->[点燃] 2N2 + 6H2O}$$

# 铵盐

- 晶体
- 能溶于水
- 受热易分解，放出氨

与碱反应：

$$\ce{NH4+ + OH- ->[\triangle] H2O + NH3 ^}$$

例：

$$\ce{NH4Cl + NaOH ->[\triangle] NaCl + H2O + NH3 ^}$$

所以铵态氮肥不能和草木灰等化肥一起使用。

可以利用这个特性**检验** $\ce{NH4+}$.
