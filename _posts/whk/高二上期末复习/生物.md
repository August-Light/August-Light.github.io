# 1.3 基因选择性表达导致细胞的差异化

## 基因选择性表达 -> 细胞分化

- **选择性表达**：机体或细胞根据环境的变换作出响应，关闭或启动一些基因的表达
- **胚胎发育**：受精卵经过多次细胞分裂，形成胚胎
  - 受精卵第一次分裂时，胞质分裂不均等，使得两个子细胞内容物发生差异。

## 表观遗传

- **表观遗传**：有些基因中，碱基序列不改变，基因表达的过程受影响，使生物个体或细胞的表型发生可遗传的改变。主要包括以下 $3$ 种：
  - **DNA 甲基化**：在酶的催化下，一些胞嘧啶发生**甲基化**，阻碍**转录**发生，使甲基化部位的基因不能表达。
    - 与转录相关的蛋白质无法结合到启动子上。
  - **组蛋白修饰**：保持 DNA 分子结构稳定性；阻碍 RNA 聚合酶与 DNA 结合。
  - **DNA 干扰**：阻断**翻译**过程。

# 2.1 有性生殖中遗传信息通过配子传递给子代

## 减数分裂

**减数分裂**：染色体数目减半的细胞分裂方式。

减数分裂是进行有性生殖的生物在产生成熟**生殖细胞（配子）** 时进行的染色体数目减半的细胞分裂。

在减数分裂时，**染色体复制一次，细胞连续分裂两次**。

### 过程（以精原细胞为例）

[一生儿！！！](https://www.bilibili.com/video/BV1XS4y1z7hG/)

进行减数第一次分裂的是初级精母细胞。

- 间期 I
  - **DNA 复制**（$46 \times 2 = 92$），合成蛋白质。
- 前期 I
  - 核膜核仁消失。
  - **同源染色体联会**形成**四分体**。
  - （后面会聊到的**交叉互换**可能在此时发生）
- 中期 I
  - 同源染色体排列在赤道板两侧。
- 后期 I
  - 同源染色体分离，移向细胞两极；等位基因分离
  - 非同源染色体上的非等位基因自由组合。人话：不是同源染色体则互相之间独立。
- 末期 I
  - 核仁核膜重建。形成两个子细胞（**次级精母细胞**，$\frac {92} 2 = 46$）。
- 间期 II
  - 多数物种没有间期 II，少数物种间期 II 很短。
  - **不进行 DNA 复制**。
- 前期 II、中期 II、后期 II、末期 II **与有丝分裂相似**。
  - 最终形成**精细胞**（$\frac {46} 2 = 23$），进一步通过一些过程成为**精子**。
  - 区别？
    - 有丝分裂**有同源染色体**，减数第二次分裂**没有同源染色体**。在减数第二次分裂中，实际上分离的是**姐妹染色单体**。

## 上课板书

> 减数分裂产生染色体数目减半的配子。
> - 范围：进行有性生殖的生物（的生殖细胞）
> - 时间：原始的生殖细胞形成成熟的生殖细胞
> - 特点：细胞连续分裂两次而 DNA 在整个分裂过程中只复制一次的细胞分裂方式。
> - 结果：新产生的成熟生殖细胞中染色体数目减少一半。

# 2.2 亲代基因传递给子代遵循特定规律

- **性状**：细胞或个体具有的可观察、测定的形态特征或生理特性。e.g. 豌豆的颜色
  - **表型**：性状的表现。e.g. 这株豌豆的颜色是绿色
  - **相对性状**：同种生物同一性状不同表型。e.g. 黄色 or 绿色

## 豌豆杂交实验

### 实验现象

- 用纯种高茎豌豆和纯种矮茎豌豆（亲本 $P$）杂交，得到的子代植株（$F_1$）**都表现为高茎**。
- $F_1$ 自交得到的子二代（$F_2$）**又出现了高茎和矮茎两种表型**。该现象称为**性状分离**。
  - 高茎和矮茎比例为 $3:1$。

### 提出假设

- 生物的性状由基因决定，这些基因独立，不会融合或在传递中消失。
- 每一对相对性状受**一对基因**控制。
  - 这对基因属于**等位基因**，一个为**显性**，一个为**隐性**。
- 体细胞中的一对等位基因中，**一个来自父方，一个来自母方**。
  - 在形成配子（减数分裂）的过程中，彼此分开，**每个配子只能得到其中的一个**。

### 解释实验

- 高茎豌豆含有 $TT$ 基因，矮茎豌豆含有 $tt$ 基因。
- 含 $T$ 基因的配子与含 $t$ 基因的配子结合形成 $F_1$。因为 **$T$ 基因对于 $t$ 基因为显性**，$F_1$ 植株表现为高茎。
- $F_1$ 产生配子时，$T$ 和 $t$ 分开，雌雄配子中仅含有 $T$ 或 $t$ 中的一个。随机结合，$F_2$ 中 $TT : Tt : tt = 1:2:1$，即高茎与矮茎之比为 $3:1$。
  - $TT, Tt, tt$ 称为**基因型**。基因型 $TT, tt$ 称为**纯合子**或**纯合体**，基因型 $Tt$ 称为**杂合子**或**杂合体**。

### 验证解释

- 将 $Tt$ 和 $tt$ 测交，得到接近 $1:1$ 比例。

**分离定律**：在生物体细胞中，控制同一性状的等位基因**成对存在，不相融合**；在减数分裂产生配子时，等位基因发生分离，随配子独立地遗传给子代。

## 在豌豆杂交实验中研究多种性状

如果要研究多种相对性状，那怎么办？

- 子叶颜色：黄色 / 绿色
- 种子形状：圆粒 / 皱粒

### 实验现象

- 亲本 $P$：**黄色圆粒**和**绿色皱粒**
- 子代 $F_1$：全部**黄色圆粒**
- 子二代 $F_2$：黄色圆粒、黄色皱粒、绿色圆粒、绿色皱粒比例为 $9:3:3:1$。

### 新的假设

- 控制不同性状的基因互不干扰
  - 独立分离和随机组合
  - 各种配子结合的机会均等

### 解释实验

- $P$：黄色圆粒 $YYRR$；绿色皱粒 $yyrr$
- $F_1$：黄色圆粒 $YyRr$
- $F_2$：懒得写了。

**自由组合定律**：位于不同染色体上的两对或两对以上非等位基因，在配子形成过程中，同一对基因彼此分离，分别进入不同的配子；不同对的基因可自由组合。

# 2.3 性染色体上的基因传递与性别相关联

## 最常见的性别决定类型——XY 型

- 人类的染色体
  - 常染色体：$1$ 到 $22$ 号染色体，每对中的两条同源染色体（一条来自父方，一条来自母方）形态、大小相同。
  - 性染色体：两条染色体形态、大小不同。男女性性染色体组成不同。
- **性别决定**：生物体合子发育成某个性别个体的预决定。
- **XY 型**是最常见的性别决定类型。
  - 雄性 XY；雌性 XX。

|  | 雄性♂ | 雌性♀ |
| - | - | - |
| XY 型 | XY | XX |
| ZW 型 | **ZZ** | **ZW** |

## 摩尔根的果蝇杂交实验

> [来源](https://zh.moegirl.org.cn/%E8%AF%BE%E6%96%87%E6%A2%97)
>
> ……它注定要成为科学史上著名的动物。摩尔根精心照料这只果蝇。在自己的第三个孩子出生时，摩尔根赶到医院，他妻子的第一句话竟是：“那只白眼果蝇怎么样了？”摩尔根的第三个孩子长得很好，但那只果蝇却很虚弱。摩尔根晚上把它带回家中，让它呆在床边的一个瓶子里，白天又把它带回实验室。在实验室，**它临死前抖擞精神，与一只红眼果蝇交配**，把突变基因传了下来。

果蝇是一种 XY 型性别决定的昆虫。

### 实验 ①

- $P$：白眼♂果蝇，红眼♀果蝇（纯合子）
- $F_1$：全部红眼
- $F_2$：♀蝇全部红眼，♂蝇红眼白眼 $1:1$

但是分离定律表明若红眼白眼和性别没关系，$F_2$ 应该是红眼白眼 $3:1$，这意味着红眼白眼和性别有关系。

（摩尔根的解释：$(X^W X^W) \times (X^w Y) \to (X^W X^w), (X^W Y)$）

### 实验 ②

- $P$：白眼♂蝇，①中 $F_1$ 的红眼♀蝇（即杂合子）
- $F_1$：无论雌雄，都是 $1:1$。

这意味着① $F_1$ 的红眼♀蝇是杂合体。

（摩尔根的解释：$(X^W X^w) \times (X^w Y) \to$ 四种组合）

### 实验 ③

与实验 ① 为正反交，即把实验 ① 中父母的性别对调。

- $P$：白眼♀蝇，红眼♂蝇
- $F_1$：♀蝇全部红眼，♂蝇全部白眼。
- $F_2$：无论雌雄，都是 $1:1$。

（摩尔根的解释：$(X^w X^w) \times (X^W Y) \to (X^W X^w), (X^w Y)$）

### 实验设计思想。

- 经过实验可知，红眼是显性，白眼是隐性。在知道这个结论的基础上，尝试所有的红白眼组合。
  - ①：白眼♂蝇，纯合子红眼♀蝇
  - ②：白眼♂蝇，杂合子红眼♀蝇
  - ③：红眼♂蝇，白眼♀蝇

### 摩尔根的结论

**控制眼色的基因 $W / w$ 应该在 $X$ 染色体上**。
- $W$ 显性：$X^W X^W, X^W X^w, X^W Y$ 红眼。
- $w$ 隐性：$X^w X^w, X^w Y$ 白眼。

**伴性遗传**：控制性状的基因传递与性别相关联。

## 伴性遗传病

- 伴 X 隐性遗传病（$X^w X^w$，$X^w Y$）
  - 母病子必病；女性患者的父亲必患病
  - $P_男 > P_女$（因为男的只要有一个就寄了，女的要两个全是才会寄）
- 伴 X 显性遗传病（$X^W X^W, X^W X^w, X^W Y$）
  - 父病女必病
  - $P_男 < P_女$（因为男的只能从母亲获得 $W$，但是女的可以从母亲和父亲两条路获得 $W$）
- 伴 Y 遗传病（$X Y^W$）
  - 父亲的致病基因只传递给儿子
- 常染色体遗传病
  - $P_男 = P_女$

# 3.1 基因重组造成变异的多样性

## 染色体自由组合可导致基因重新组合

- 举个例子，一种性状有 A 和 a $2$ 个基因，另一种性状有 B 和 b $2$ 个基因，总共就有 $2^2 = 4$ 种组合。性状更多组合更多。
- 进一步地，可能出现表型变异。
- 应用：**杂交育种**

## 染色体交叉互换导致基因重组

咋又是摩尔根，摩尔根得了 MVP。

### 实验

摩尔根又来倒腾果蝇了。

- $P$：纯种黑身残翅♂，纯种灰身长翅♀
- $F_1$：灰身长翅
- 用【$F_1$ 灰身长翅♀】和【纯种黑身残翅♂】测交得到 $F_2$：
  - 自由组合定律预测：四种组合 $1:1:1:1$
  - 实际结果：和 $P$ **相同的个体数远多于预期**，和 $P$ **不同的个体数远少于预期**。

### 推理

首先 $F_2$ 有很多和 $P$ 一样的，这意味着很可能这两种相对性状是“捆绑”的：**控制【长翅 / 残翅】【灰身 / 黑身】的基因位于同一对同源染色体上**。

但是好像也不对，这无法解释少数的和 $P$ 不同的个体。

于是我们再引入一个机制：**染色体交叉互换**。

$F_1$ 原本长这样 `B b Vg vg`：
- 按照正常方式产生配子，则有 `B Vg` 和 `b vg` 这两种（因为是绑定的）。
- 但是若发生了交叉互换，则可以产生 `B vg` 和 `b Vg` 这两种。

这就导致了**基因重组**。

不是每一个细胞都发生染色体交叉互换，每一次染色体交叉互换一般只涉及 $2$ 条非姐妹染色单体而不是 $4$ 条。因此交叉互换是一个小概率事件，解释了为什么会【远少于预期】。
