## 一、整体概括（抓主线）

14—18 世纪，随着**资本主义生产关系的萌芽和发展**，欧洲在思想文化领域经历了一系列深刻变革：

* **文艺复兴**：以“复兴”古典文化为形式，实质上宣扬**人文主义**，冲击中世纪以神为中心的思想体系；
* **宗教改革**：从宗教领域反对天主教会的权威，强调个人信仰与民族教会，进一步解放思想；
* **近代科学革命**：用实验、理性和数学方法认识自然，动摇宗教神学对世界的解释权；
* **启蒙运动**：以理性为核心，系统批判封建专制和宗教迷信，为资产阶级政治制度提供理论依据。

这些运动在思想上**逐步瓦解封建神权与专制统治的合法性**，为资本主义制度的确立和资产阶级革命奠定了思想基础。

## 二、系统整理（按“运动—核心思想—代表人物—代表观点/著作”）

### （一）文艺复兴（14—17 世纪初）

**运动性质**

* 资产阶级新文化运动
* 形式上复兴古典文化，实质上反封建、反神权

**核心思想**

* **人文主义**

  * 以人为中心，而非以神为中心
  * 肯定人的价值、尊严和理性
  * 追求现世幸福，反对禁欲主义

**代表人物及作品**

| 领域     | 人物    | 代表作 / 特点                |
| ------ | ----- | ----------------------- |
| 文学     | 但丁    | 《神曲》，批判教会腐败             |
| 文学     | 彼特拉克  | 人文主义文学奠基者               |
| 文学     | 薄伽丘   | 《十日谈》，讽刺教会              |
| 美术     | 达·芬奇  | 《最后的晚餐》《蒙娜丽莎》，科学精神与艺术结合 |
| 美术     | 米开朗琪罗 | 《大卫》，突出人体之美             |
| 美术     | 拉斐尔   | 《雅典学院》，理性与和谐            |
| 文学（后期） | 莎士比亚  | 《哈姆雷特》《李尔王》，人性复杂性       |

**历史作用**

* 解放思想
* 冲击中世纪神学世界观
* 为资本主义发展提供精神动力

### （二）宗教改革（16 世纪）

**运动背景**

* 天主教会腐败、盘剥严重
* 资产阶级和民族国家反对教廷控制

**核心思想**

* 信仰得救（因信称义）
* 上帝面前人人平等
* 建立**民族教会**，削弱教会特权

**代表人物与主张**

| 国家  | 人物    | 代表观点           |
| --- | ----- | -------------- |
| 德意志 | 马丁·路德 | 《九十五条论纲》，反对赎罪券 |
| 瑞士  | 加尔文   | 加尔文派，新教重要分支    |
| 英国  | 英王领导  | 建立国教（圣公会）      |

**历史作用**

* 动摇天主教会统治
* 促进思想解放
* 推动民族国家形成
* 有利于资本主义成长

---

### （三）近代科学革命（16—17 世纪）

**运动性质**

* 自然科学领域的革命
* 以理性和实验取代神学解释自然

**核心思想 / 方法**

* 尊重事实与经验
* 观察、实验、分析、归纳
* 科学理性思维的确立

**代表人物及观点**

| 人物  | 领域  | 代表观点          |
| --- | --- | ------------- |
| 哥白尼 | 天文学 | 日心说，否定地心说     |
| 牛顿  | 物理学 | 万有引力定律，经典力学体系 |

**历史作用**

* 动摇宗教神学权威
* 推动思想解放
* 为启蒙运动提供科学基础

---

### （四）启蒙运动（17—18 世纪）

**运动性质**

* 资产阶级反封建、反专制的思想解放运动
* 法国为中心，影响欧洲和北美

**核心思想**

* **理性主义**
* 天赋人权、自由、平等
* 法治与权力制衡
* 相信进步和教育

**代表人物及思想**

| 人物    | 国家 | 代表思想 / 著作  |
| ----- | -- | ---------- |
| 孟德斯鸠  | 法国 | 三权分立       |
| 伏尔泰   | 法国 | 开明君主制      |
| 卢梭    | 法国 | 主权在民       |
| 洛克    | 英国 | 天赋人权       |
| 亚当·斯密 | 英国 | 自由竞争，《国富论》 |
| 康德    | 德国 | 理性、自主、自由平等 |

**历史作用**

* 为资本主义制度提供理论基础
* 推动美国独立战争、法国大革命
* 成为民族解放的重要思想武器
