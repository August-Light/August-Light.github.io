# Models of Memory

- **Sensory Memory**
  - e.g. 挥舞烟花棒时看到的残影
- **Short-term Memory**
  - e.g. 记住今天早上把自行车停在哪里
- **Long-term Memory**
  - e.g. 记住你的电话号码

## Short-term Memory

- **Selective Attention**：决定哪些信息值得记住，哪些不值得
- **Cocktail party effect**：即使存在多种刺激，甚至有些刺激可能更响亮或更突出，我们仍然能够准确地专注于某一个对话。
- **Chunking**：用首字母方便记忆。

## Long-term memory

- **Episodic memory**：记住事情发生的顺序
- **Semantic memory**：一般常识
  - e.g. 篮球是橙色的
- **Procedural memory**：掌握技能

---

- **Explicit memories**：刻意回忆想起的内容
- **Implicit memories**：自然而然浮现的内容

# Levels of processing

- **Shallow processing**：识别物体的物理特征
  - e.g. 看到了车上的线条
- **Intermediate processing**：认出物体
  - e.g. 认出车
- **Deep processing**：想到了相关联的东西
  - e.g. 认出这是法拉利

# Memory Retrieval

- **Recognition**：认出熟悉事物
- **Recall**
  - Chris 称，书上说 recall 需要外部提示，但是他不同意。
- **Serial position effect**
  - **Primacy effect**：更容易记住列表开头的内容
  - **Recency effect**：更容易记住列表末尾的内容
    - 【说文解字】recent
- **Semantic (语义) network theory**：在现有记忆的基础上构建新的记忆。
- **Tip-of-the-tongue phenomenon**：话卡在嘴边说不出来。
- **Flashbulb memories**：与重要事件相结合的普通记忆。
  - e.g. 清楚地记得 911 那天自己在干什么。
- **Mood-congruent memory**：更容易回忆起和目前情绪相似的记忆。
  - e.g. 快乐的时候更容易想起快乐的记忆。
- **State-dependent memory**：记忆的形成有可能与我们的精神状态有关。
  - e.g. 喝了一堆咖啡后学的东西，更容易在喝了一堆咖啡后想起来。
- **Constructed memories**：被压抑了很长时间后随机出现的记忆。
  - 很多是被诱导性问题构造出来的虚假记忆。

# Forgetting

- **Re-learning effect**：学过忘了的东西更容易再次学会。
- **Interference** 也是一种常见的造成遗忘的方式。
  - **Retroactive interference**：忘了老的记住新的。
    - 【说文解字】retro，后面的新的
  - **Proactive interference**：忘了新的记住老的。
    - 【说文解字】pro，前面的老的

# Memory formation

- **anterograde amnesia (顺行性遗忘症)**：无法形成新的记忆，但老的还记得。
  - 【说文解字】antero 向前的（与 retro 向后相对）
    - 因此 retrograde amnesia 就是把老的事情忘了。
- **Long term potentiation**：重复刺激使神经元连接增强。
  - 【说文解字】potent 强大的，有能力的

# Language

- **Phonemes (音素)**
  - e.g. cat 有 $3$ 个音素：/k/ /æ/ /t/
- **Morphemes (语素)**
  - e.g. deconstructed 有 $4$ 个语素：de con struct ed
- **Syntax (语法)**

## Language acquisition

- 小孩 4 个月大说音素，1 岁说单词，1 岁半组合单词。
- 语法错误不可避免。小孩可能把 bite -> bit 的变形搞错成 bite -> bited，这被称为 **overgeneralization**。

---

- 一帮人认为语言的习得通过 operant conditioning 和 shaping。
- 另一帮人认为我们天生有 **language acquisition device (语言习得机制)**，意味着我们在小时候可以快速学习语言。
- 大多数心理学家认为答案在这两者之间。

---

- **linguistic relativity hypothesis**：我们使用词语的方式会影响我们的观念。
  - e.g. 在英语和汉语中“雪”只有一个词，在 Yupik 语中有很多表示不同的雪的词，比如“积雪”“落雪”“可融化的雪”。

# Algorithm & Heuristics

- **Algorithm (算法)**：一套规则，遵循步骤就能得到正确答案。
- **Heuristics (启发式方法)**：帮助我们得到准确结论的基本但不完美的规则，不一定准。
  - **Availability heuristic**：根据“例子在脑中浮现得有多容易”来判断概率。
    - e.g. 飞机失事好可怕，觉得坐飞机比开车更危险（实际上显然开车事故概率更高！）。
  - **Representative heuristic**：根据“是否符合刻板印象 / 原型”，来判断概率或类别。
    - e.g. (ChatGPT) 一个读书安静不善交际的人更可能是农民还是图书管理员？很多人会觉得是图书管理员。（实际上农民比读书管理员多得多，基数大！）

# Problems with solving problems

- **Rigidity**：抵制改变的倾向。
  - **Functional fixedness**：只能想到物体主要用途，除了主要以外的其他用途想不到。
- **Confirmation bias**：倾向于寻找并接受那些证实我们已有信念的信息，而忽略那些与我们已有信念相悖的信息。
- **Framing**：使用特定的表达方式。语言的艺术
  - e.g. “我答对了 80%” instead of “我答错了 20%”。

# Creativity

- **Convergent thinking**：对一个问题想一个解
- **Divergent thinking**：对一个问题想很多创造性解法
