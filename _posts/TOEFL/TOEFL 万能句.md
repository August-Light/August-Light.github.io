# Topic Sentence 怎么写？

- **If** + 抄题，sb can ... / **By doing** something, sb can
- **It is adj. for** sb to do ...
- **Doing** something can make sb do / **allow**s sb to do
- **There be** ...

# 结尾句怎么写？

- **This is** very + adj. + for something
- ..., **which is** very + adj. + for something
- It’s very + adj. + for something

这里的 adj. 和 something 怎么填？
- adj.: **helpful / important / useful / beneficial**
- something: **friendships / future / development / health**

# Because / so 用太多了怎么办？

- since
- as
- probably because
- mainly because
