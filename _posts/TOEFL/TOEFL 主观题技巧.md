# 论证的基本方法

- 先问为什么
- 从左往右扩写
  - 在开头尝试补原因
  - 举例，对每个名词或动词尝试举例
  - 在结尾补充结果

# 怎么想第二个理由？

谜底就在谜面上！

找到问题中的所有对象，考虑对于他们的好处和坏处。

# Topic Sentence 怎么写？

- If + 抄题，sb can ... / by doing sth , sb can
- It is adj for sb to do ...
- Doing sth can make sb do / allows sb to do
- There be ...

# 结尾句怎么写？

- This is very + adj. + for sth.
- Which is very + adj. + for sth.
- It’s very + adj. + for sth.

这里的 adj. 和 sth. 怎么填？
- adj.: helpful / important / useful / beneficial
- sth.: friendships / future / development / health

# 注意点

## 主语统一

问 you 就答 I，问 people 就答 they，不要 you。

## 话说清楚

- learn effectively ×
  - have a deeper understanding

## 直接进主题

- 让学生学得更好 ×
  - 学得更 engaging，因此学得更好 √
  - 然后就方便举例子了！可以举出各种好玩的东西，说 motivation
